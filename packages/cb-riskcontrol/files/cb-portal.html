核心防护矩阵

⚡

硬件级 Nftables 拦截

彻底抛弃陈旧的 iptables。系统底层 强制 TTL=128 伪装（模拟 Windows 办公），并在 OOB 层拦截所有 WebRTC (STUN/TURN) 协议端口，100% 杜绝浏览器真实 IP 穿透。

🕒

自动时区与流缓存加速

守护进程精准解析节点 Timezone (时区)，在毫秒级自动对齐系统时间。配合 HW/SW Flow Offloading 软硬双重加速，即便跑满千兆带宽双核 CPU 占用依旧低于 5%。

🛑

流量锁 (Kill Switch)

业界独创的 Nftables 秒级流量锁。内置超高频连通性检测，一旦侦测到代理节点掉线，瞬间物理阻断所有内网数据包转发，宁可断网也绝不允许真实 IP 发生一次外包泄露！

👻

MAC 随机化防追踪

您的物理 MAC 地址（BSSID）也是高危追踪特征！CB-Shield V3.0 会在每一次路由器重启时，为您生成并分配 全套全新的合法私有 MAC 地址，彻底切割历史关联设备特征。

🛡️

DNS 强劫持回源

还在担心客户端写死了 `8.8.8.8` 而绕过代理导致漏包？V3.0 通过专有 DNAT 接管，将所有 53 端口流量物理重定向至本地加密代理通道，实现绝对的数据匿名。

🎥

SQM/Cake 视频流整形

专为 TikTok 直播带货 / YouTube 高清上传 优化。独特的 qdisc 调度算法确保电商数据包永远具备最高发包优先级，彻底消灭局域网拥堵带来的 Bufferbloat 延迟刺突。
