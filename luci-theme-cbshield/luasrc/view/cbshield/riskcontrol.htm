📖 风控说明

风险评分规则

  • 代理 IP：+50 分 — 检测到 IP 经过代理服务器转发
  • 托管/数据中心 IP：+30 分 — IP 属于云服务器/数据中心
  • 可疑组织：+20 分 — ISP/组织名包含 VPN、Proxy 等关键词

风险等级

  • 🟢 低风险 (0-29分)：IP 安全
  • 🟡 中风险 (30-49分)：需注意
  • 🟠 高风险 (50-79分)：建议切换 IP
  • 🔴 极高风险 (80-100分)：立即处理

👑 V2.2 黄金版 - 核心级防护机制 (原生运行)

  • TTL 指纹伪装：强制外网出口 TTL 为 128 (模拟标准 Windows 办公环境)，消除路由器指纹。
  • WebRTC 物理阻断：底层屏蔽 STUN/TURN 全系端口，100% 防止浏览器真实 IP 穿透泄漏。
  • 极速时区对齐：守护进程根据代理 IP 精准时区自动对齐系统，消除跨区时差指纹。
  • 流量锁 (Kill Switch)：基于 Nftables 开发，一旦检测到代理掉线，瞬间物理阻断内网转发，杜绝裸连！
